DÍA 2: PROFUNDIZACIÓN

Estructura del Razonamiento y Simbolización

Desde la estructura atómica de una idea hasta la formalización matemática. Entendiendo cómo las premisas construyen la verdad.

1. Anatomía de la Proposición

Proposiciones Simples

El átomo del pensamiento. Expresan una sola idea indivisible.

Principio Cartesiano

"Naturalezas simples: Claras y Distintas."

  • Clara: Se entiende sin confusión.
  • Distinta: No se mezcla con otras ideas.
let p = "El número 2 es par";

Proposiciones Compuestas

Moléculas lógicas. Formadas por varias simples + conectivos.

Premisas: Proposiciones que aportan información base.
Inferencia: Proceso Mental que conecta premisas.
Conclusión: Resultado lógico. No añade info nueva, explicita lo oculto.

2. El Motor Lógico: Silogismos

Un silogismo es la forma clásica del razonamiento: P + P = C

¿Qué es razonar?

Es el proceso algorítmico mediante el cual la mente combina proposiciones (inputs) para obtener una conclusión válida (output).

Nota: Si las premisas son verdaderas y la estructura es válida, la conclusión debe ser verdadera.
INPUTS (Premisas) PROCESO
P1 Todos los humanos son mortales.
P2 Sócrates es humano.
INFERENCIA EN PROCESO...
Conectando "Sócrates" con "Mortal"
C SÓCRATES ES MORTAL.
René Descartes

El Método

Este es el nacimiento de la Descomposición Algorítmica y la Lógica Simbólica. Para evitar las trampas del lenguaje humano, convertimos las ideas en símbolos matemáticos precisos.

Diccionario Lógico-Matemático

Conectivos Lógicos
Símbolo Nombre Lectura

&lor; Disyunción "... o ..."

3. Traducción y Axiomas

Ejemplo 1 (Naturales)

∀x ∈ ℕ : x > 0

"Para todo x que pertenece a los números naturales, x es mayor que cero."
Ejemplo 2 (Existencial)

∃x ∈ ℝ : x² = 4

"Existe un número real x tal que x al cuadrado es igual a 4."
Ejemplo 3 (Implicación)

∀x (x ∈ A → x ∈ B)

"Para todo x, si x pertenece al conjunto A, entonces pertenece al conjunto B."
Axioma

Verdad evidente que no necesita demostración. Es el punto de partida (ej. "El todo es mayor que la parte").
