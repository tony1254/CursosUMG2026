8. Taller de Tablas de Verdad (Álgebra Booleana)

Resuelve a mano respetando la jerarquía de operaciones (Paréntesis → Negación → Conjunción/Disyunción → Condicional → Bicondicional).

  • ( p ∨ q ) → ( q ∨ p )
  • [(p → q) ∧ (q → r)] → (p → r)
  • [(p → ¬q) ∧ (¬p → ¬r) ∧ (¬r → s)] → s
  • [(¬p ∨ q) ∨ (p ∧ q)] → [(¬p ∨ q) ∨ ¬p]
  • [(p ∨ ¬q) → (p → q)] → [(¬p → q) ∨ ¬p] ∨ ¬p
  • (p ∧ q → p ∨ ¬r) → ¬(¬q ∨ ¬r) ∧ r
  • (¬p ∧ q → p ∨ r) ↔ ¬(¬q ∨ ¬r) ∧ r
  • (p ∨ ¬q → p ∧ r ) ↔ [ ¬(¬ q ∨ ¬ r) ∨ ( r → ¬ q) ]
  • (¬p ↔ q ∨ ¬r) ∧ ¬(¬p ↔ q ∨ ¬r)
